# GRIMOIRE

**Enchant your workflow**

GRIMOIRE is a task management application inspired by the Kanban method. Organize your projects on visual boards and move forward without distractions.

## Technologies

- **Frontend**: React + TypeScript
- **Backend**: Node.js + Express + MongoDB
- **Authentication**: JWT
- **Styling**: TailwindCSS (optional)

## Installation

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Features

- Create and manage boards.
- Drag and drop cards between lanes.
- User authentication with JWT.
- Data persistence in MongoDB.
